## Minimum Sizes

| Format | Minimum Width |
|--------|---------------|
| Full Logo | 120px |
| Icon Only | 24px |
| Favicon | 16px |

## Usage Rules

### DO
- Use provided SVG files
- Maintain aspect ratio
- Ensure adequate contrast with background
- Use appropriate color treatment for background

### DON'T
- Stretch or distort
- Add effects (shadows, gradients, outlines)
- Place on busy backgrounds without contrast
- Rotate or skew
- Change colors outside approved treatments
- Crop or partially hide

## File Formats

| Format | Usage |
|--------|-------|
| SVG | Web, scalable, primary |
| PNG | Raster needs, transparency |
| ICO | Favicons (if needed) |
