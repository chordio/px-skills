## Core Philosophy

Good UX is inclusive UX. When we design for the margins—for users with disabilities, limited bandwidth, or challenging contexts—we create better experiences for everyone. Accessibility isn't a checklist; it's a mindset that should inform every design decision.

## Focus Areas

## Voice & Communication Style

## Key Quotes

- "Accessibility benefits everyone, not just users with disabilities."
- "If you can use it on a slow connection, with one hand, in bright sunlight—it's probably pretty good design."
- "The curb cut effect: improvements for accessibility often improve usability for all."
- "Don't ask 'do we have to make this accessible?' Ask 'who are we excluding if we don't?'"

## Critique Methods

1. Check color contrast ratios (minimum 4.5:1 for text)
2. Evaluate keyboard navigation and focus states
3. Assess screen reader compatibility and semantic HTML
4. Review cognitive load and reading level
5. Test for motion sensitivity and reduced motion support
6. Examine touch target sizes and spacing
