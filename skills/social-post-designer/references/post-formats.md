# Social Post Format Specifications

Detailed specifications for each post format type supported by the social-post-designer skill.

## Text-Only Posts

### Best Use Cases
- Quick tips and insights
- Thought-provoking questions
- Announcements and updates
- Quotes and inspiration
- Polls and engagement questions
- Behind-the-scenes stories

### Structure
1. **Hook** (First 3-5 words): Must stop the scroll
2. **Body** (2-4 sentences): Delivers value or story
3. **Engagement Element**: Question, poll, or discussion starter
4. **CTA** (if applicable): Link, comment prompt, or save reminder

### Length Guidelines
- **LinkedIn**: 150-300 words optimal
- **Twitter/X**: 1-2 tweets (280 characters each)
- **Instagram**: 125-150 words (optimal engagement)
- **Facebook**: 40-80 words (higher engagement)
- **TikTok**: 50-100 words (caption, not video text)

### Formatting Tips
- Use line breaks for readability
- Emojis sparingly (1-2 max for professional platforms)
- Hashtags at end (3-5 for LinkedIn, 5-10 for Instagram)
- Questions in separate lines for emphasis

### Example Structure
```
[HOOK - Question/Statement/Number]

[Value delivery - 2-3 sentences]

[Engagement driver - question or call to action]

[Hashtags if applicable]
```

## Image Posts

### Best Use Cases
- Product showcases
- Infographics and data visualization
- Behind-the-scenes photos
- Quote graphics
- Before/after comparisons
- Screenshots and UI showcases

### Dimensions by Platform
- **LinkedIn**: 1200 x 627px (1.91:1) or 1080 x 1080px (square)
- **Twitter/X**: 1200 x 675px (16:9) or 1080 x 1080px (square)
- **Instagram Feed**: 1080 x 1080px (square) or 1080 x 1350px (4:5)
- **Instagram Story**: 1080 x 1920px (9:16)
- **Facebook**: 1200 x 630px (1.91:1)
- **TikTok**: 1080 x 1920px (9:16)

### Copy Structure
1. **Hook**: References or complements the image
2. **Context**: Explains what the image shows
3. **Value**: Key takeaway or insight
4. **CTA**: Action related to the visual

### Visual-Text Integration
- Copy should not repeat what's visible
- Copy should add context or insight
- Use copy to guide attention to key visual elements
- Create curiosity gaps between visual and text

### Example Structure
```
[HOOK that references the image]

[What you're seeing / Context]

[Key insight or takeaway]

[CTA - what to do next]

[Hashtags]
```

## Image Carousel Posts

### Best Use Cases
- Step-by-step tutorials
- Multiple product features
- Story sequences
- Before/during/after narratives
- Comparison sets
- Multi-point explanations

### Dimensions
- **LinkedIn**: 1080 x 1080px per slide (square)
- **Instagram**: 1080 x 1080px per slide (square)
- **Facebook**: 1080 x 1080px per slide (square)
- **Twitter/X**: 1080 x 1080px per slide (square)

### Slide Count
- Optimal: 3-7 slides
- Maximum engagement: 5 slides
- Minimum: 3 slides (to justify carousel format)

### Narrative Structure
1. **Slide 1**: Hook slide - most compelling visual/statement
2. **Slides 2-N-1**: Value delivery - step-by-step or feature-by-feature
3. **Final Slide**: CTA slide - clear action with visual emphasis

### Copy Structure
- **Opening**: Sets up the carousel journey
- **Body**: Can reference specific slides ("Slide 3 shows...")
- **Closing**: Strong CTA that leverages the full carousel

### Visual Consistency
- Maintain consistent style across all slides
- Use same color palette and typography
- Create visual flow between slides
- Ensure each slide works standalone and in sequence

### Example Structure
```
[HOOK - introduces the carousel journey]

[Swipe through to see... / Check out these 5 steps...]

[Key insight from the full set]

[CTA - swipe to see more / save this post]

[Hashtags]
```

## Video Posts

### Best Use Cases
- Product demonstrations
- Tutorials and how-tos
- Testimonials and case studies
- Behind-the-scenes content
- Animated explanations
- User-generated content showcases

### Dimensions by Platform
- **LinkedIn**: 1920 x 1080px (16:9) or 1080 x 1920px (9:16)
- **Twitter/X**: 1280 x 720px (16:9) or 720 x 1280px (9:16)
- **Instagram Feed**: 1080 x 1080px (square) or 1080 x 1920px (9:16)
- **Instagram Reels**: 1080 x 1920px (9:16)
- **TikTok**: 1080 x 1920px (9:16)
- **Facebook**: 1280 x 720px (16:9) or 720 x 1280px (9:16)

### Duration Guidelines
- **LinkedIn**: 30-90 seconds
- **Twitter/X**: 15-60 seconds
- **Instagram Reels**: 15-90 seconds
- **TikTok**: 15-60 seconds (optimal: 21-34 seconds)
- **Facebook**: 15-60 seconds

### Copy Structure
1. **Hook**: Creates curiosity about the video
2. **Preview**: Teases what they'll see/learn
3. **Value Promise**: What they'll gain from watching
4. **CTA**: Watch, save, share, or comment

### Visual-Text Integration
- Copy should complement, not describe the video
- Use copy to set up the video's narrative
- Create curiosity gaps that video fills
- Reference specific moments if relevant

### Thumbnail Considerations
- First frame should be compelling (hook frame)
- Include text overlay if needed for context
- Ensure readability at small sizes
- Match brand visual style

### Example Structure
```
[HOOK - creates video curiosity]

[What you'll see in this video / The problem this solves]

[Value promise - what they'll learn/gain]

[CTA - watch now, save for later, share with someone who needs this]

[Hashtags]
```

## Format Selection Guide

### Choose Text-Only When:
- Message is powerful enough without visuals
- Quick, timely updates
- Discussion-starting questions
- Platform favors text (LinkedIn, Twitter)

### Choose Image When:
- Visual adds significant value
- Product or result needs showcasing
- Data or information needs visualization
- Single compelling visual tells the story

### Choose Carousel When:
- Multiple points need visual support
- Step-by-step process benefits from sequence
- Comparison or before/after narrative
- Multiple features to highlight

### Choose Video When:
- Motion/demonstration is essential
- Tutorial or how-to content
- Storytelling benefits from time
- Platform algorithm favors video (TikTok, Reels)

## Cross-Platform Considerations

### Multi-Platform Strategy
- Create format variations for each platform
- Adapt dimensions and aspect ratios
- Adjust copy length and style
- Consider platform-specific features (polls, stickers, etc.)

### Format Consistency
- Maintain core message across formats
- Adapt presentation, not content
- Use same visual assets when possible
- Ensure brand consistency
