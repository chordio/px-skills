# Design Specification: Button Component

## Overview

A button for clicking.

## Styling

- Background color: `#007bff` (blue)
- Text color: `#ffffff` (white)
- Padding: 10px 20px
- Border radius: 4px
- Font size: 14px

## States

### Default
The button is blue with white text.

### Hover
The button changes color on hover.

### Disabled
The button is grayed out when disabled.

## Variants

### Primary
Use the primary button for main actions. It's blue.

### Secondary
A lighter button for less important actions.

## Notes

- The button should look good
- Make sure it's clickable
